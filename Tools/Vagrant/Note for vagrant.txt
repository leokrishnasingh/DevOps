VM automation tool, works on top of a hypervision like vm ware or oracle virtual box.


Key Points 
-> No OS installation 
	VM images/Box 
	free box available on vagrant cloud

-> Vagrant file 
	manages all VM settings in a file 
	VM changes through vagrant file 
	provising

->Simple commands
	vagrant init boxname
	vagrant up 
	vagrant ssh 
	vagrant halt 
	vagrant destroy
